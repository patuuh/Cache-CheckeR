# Cache-CheckeR
Script that has finds urls that use caching methods. For use of future cache poisoning matters

## Usage
```bash
python3 run.py -h
```
Scan hosts from bug bounty platforms (arkadiyt bounty targets)
```bash
python3 run.py -bb
```
Scan urls from list
```bash
python3 run.py -l hosts.txt 
```
